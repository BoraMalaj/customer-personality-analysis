# Customer Personality Analysis - Clustering

## Overview
This project applies clustering techniques on the Customer Personality 
Analysis dataset from Kaggle to segment customers based on their 
demographics and purchasing behavior.

## Dataset
- Source: Kaggle - Customer Personality Analysis
- Samples: 2240 customers
- Features: 29 features
- Task: Unsupervised Clustering

## Steps Covered
1. Data Loading and Basic Information
2. Missing Values and Duplicates
3. K-Means Clustering with Elbow Method
4. DBSCAN Clustering
5. Cluster Evaluation - Silhouette Score, SSE, Elbow Method
6. Cluster Visualization and Comparison

## Libraries Used
- pandas, numpy, matplotlib, seaborn, scikit-learn
